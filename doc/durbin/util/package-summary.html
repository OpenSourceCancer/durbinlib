Package durbin.util

Basic convenience and performance utilities, including many intended for use from Groovy. These utilities typically fill holes in the JFC or Groovy JDK.

 

Class Summary
FileUtils *********************************************************************************** Utility methods to help with common file related things.
Perlism ************************************************** Make some functionality available to Groovy that is common in Perl but not implemented in Groovy by default.
Table ***************************************************************** A 2D table of objects with methods to read from file, iterate over rows or columns, and support for Groovy closures.
TableMatrix1D ************************************************* A row or a column in the table.
TableMatrix1DIterator *********************************************** An iterator for a row or a column of a table
